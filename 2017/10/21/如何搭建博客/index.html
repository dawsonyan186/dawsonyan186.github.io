如何搭建博客

使用 Hexo+GitHub 搭建博客系统

  1. 全局安装hexo-cli

  2. 进入一个安全的目录(Desktop);

  3. 使用hexo-cli初始化myBlog目录

  4. 进入myBlog目录

  5. 编写第一篇博客，开博大吉

  6. 你会看到一个 md 文件的路径(编辑这个 md 文件)

  7. 在 GitHub 上新建一个空 repo，repo 名称是「你的用户名.github.io」

  8. 编辑网站配置start _config.yml

    • 把第 6 行的 title 改成你想要的名字
    • 把第 9 行的 author 改成你的大名
    • 把最后一行的 type 改成 type: git
    • 在最后一行，与 type 平齐，加上一行 repo: 仓库地址

  9. 安装 git 部署插件

  10. 部署博客

  11. 进入github对应的 repo，打开 GitHub Pages 功能，点击预览链接，就可以看博客了
